Audience1st CSS Style Kit

Quick Instructions

This kit contains all the patron-facing pages of Audience1st, and an example generic stylesheet called generic.css. The links and buttons on the pages are inactive; the pages are only for visual reference.

To change the styling of your Audience1st installation:

  1. Edit the generic.css stylesheet until the HTML pages look the way you want. (See below for suggestions.)
  2. Save the generic.css stylesheet under some name of your choice on your venue's public Web server and note the URL. This must be a fully-qualified URL, i.e. everything beginning with http://.
    3. Make sure the file is publicly readable: you should be able to type the URL into a Web browser and get back the stylesheet.
  3. Login as Administrator on your Audience1st installation and visit the URL http://www.audience1st.com/VENUE/options where VENUE is replaced with your venue's assigned Audience1st pathname.
  4. Under External Web Assets, change the value of the External CSS stylesheet property to the URL of your stylesheet.

Resources/Suggestions

You should have some knowledge of CSS and XHTML to do this work. If you'd rather have a professional work with you, contact us for help.

  • We recommend using a developer add-on such as Firebug for the Firefox browser or the IE Developer Toolbar for Microsoft Internet Explorer.
  • We recommend using a CSS stylesheet editor such as CSSEdit for Mac OS X or TopStyle for Windows.
  • For inspiration, check out the CSS Zen Garden pages.
  • Reference materials can be found on the official CSS home page of the WWW Consortium.
  • There are many great books on CSS design, but our local experts particularly recommend these titles:
    • Head First HTML with XHTML and CSS as a good basic introduction
    • Bulletproof Web Design: Improving flexibility and protecting against worst-case scenarios with XHTML and CSS as a good collection of more advanced tips and techniques
